# Добро пожаловать в мое портфолио проектов по анализу данных!
В репозитории собраны проекты, демонстрирующие мои навыки работы с данными на Python. Каждый проект представляет собой Jupyter Notebook с подробным анализом и визуализацией данных.

### Содержание портфолио
1. **Проекты по A/B-тестированию**
* ab_test_knigi_i_sait.ipynb - A/B-тест для анализа активности пользователей книжного сервиса и оценка эффективности новой версии сайта магазина с помощью A/B-теста
* ab_test_user_activity.ipynb - Исследование пользовательской активности в развлекательном приложении с функцией «бесконечной» ленты с помощью A/B-теста
2. **Аналитические проекты**
* pd_games.ipynb - Анализ игровой индустрии с использованием pandas
* pd_np_plt_sns_phik_startapy.ipynb - Исследовательский анализ исторических данных о финансировании стартапов с применением статистических методов
* pd_plt_sns_phik_obshepit.ipynb - Исследование рынка общественного питания Москвы
* pd_plt_st_samokaty.ipynb - Анализ данных сервиса проката самокатов
### Используемые технологии
В проектах применяются следующие библиотеки Python:

* pandas (pd) - для обработки и анализа данных
* numpy (np) - для математических операций
* matplotlib (plt) - базовая визуализация
* seaborn (sns) - расширенная визуализация
* scipy.stats (st) - статистические методы
* phik - для корреляционного анализа
### Структура проектов
Каждый проект содержит:

* Импорт необходимых библиотек
* Загрузка и предобработка данных
* Исследовательский анализ данных (EDA)
* Визуализация результатов
* Выводы и рекомендации
